# Speedometer & Pace Calculator

## Tech Stack

- **Framework:** [Flutter](https://flutter.dev/) (Dart)
- **State Management:** [BLoC](https://bloclibrary.dev/) / Provider
- **Backend & Auth:** [Firebase](https://firebase.google.com/) (Auth & Firestore)
- **Location Services:** `geolocator` plugin

---

## How to Run the App

### Prerequisites

1. Install [Flutter SDK](https://docs.flutter.dev/get-started/install).
2. Install [Android Studio](https://developer.android.com/studio) or [Xcode](https://developer.apple.com/xcode/) for device emulators.
3. Install the [FlutterFire CLI](https://firebase.flutter.dev/docs/cli/) to configure Firebase.

### Installation

1. **Clone the repository** (if you haven't already) and navigate to the project directory:
   ```bash
   cd speedometer
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the App:**
   Connect a physical device or start an emulator, then run:
   ```bash
   flutter run
   ```
